Position Overview

We are seeking talented and enthusiastic web developers to join our team at Hot Beans Web. The ideal candidate should have a strong foundation in web development technologies, including HTML, CSS, and JavaScript. This role will involve working on a variety of web projects, from client websites to internal web applications.

Key Responsibilities

  • Developing and maintaining client websites using HTML, CSS, and JavaScript.
  • Collaborating with designers and other developers to implement website features and functionality.
  • Testing and debugging code to ensure high-quality and reliable web applications.
  • Optimizing websites for performance, usability, and search engine optimization (SEO).
  • Staying up-to-date with the latest web development trends, tools, and technologies.

Qualifications

  • Bachelor's degree in Computer Science, Web Development, or a related field (or equivalent experience).
  • Proficiency in HTML, CSS, and JavaScript.
  • Experience with front-end frameworks/libraries such as Bootstrap, React, or Vue.js is a plus.
  • Strong problem-solving skills and attention to detail.
  • Excellent communication and teamwork abilities.
  • Ability to work independently and manage time effectively.
