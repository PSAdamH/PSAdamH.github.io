Junior Web Developer Position

Job Description

A junior web developer at Hot Beans Web is responsible for assisting in the development and maintenance of websites and web applications. They work under the guidance of senior developers and collaborate with the design and content teams to implement website features and functionality.

Key responsibilities of a junior web developer include:

- Writing clean and efficient HTML, CSS, and JavaScript code.

- Assisting in the development and implementation of website layouts and designs.

- Troubleshooting and debugging website issues.

- Collaborating with the design team to ensure a seamless user experience.

- Assisting in the integration of third-party APIs and services.

- Conducting website testing and optimization.

- Staying up-to-date with the latest web development trends and technologies.

A junior web developer should have a solid understanding of HTML, CSS, JavaScript, and web development principles. They should also possess good problem-solving skills and attention to detail.

Benifits

  • £25,000 per year
  • Flexible working hours
  • Opportunities for career growth and development
  • Competitive salary based on experience as well as qualifications

    Web Project Manager

    Job Description

    A web project manager coordinates web development projects, overseeing planning, execution, and delivery. They define scopes, set timelines, delegate tasks, and ensure communication among stakeholders. Managing resources and resolving challenges, they ensure projects meet objectives within budget and schedule.

    Key responsibilities of a web project manager include:

  • Planning and defining project scopes, objectives, and deliverables.
  • Setting timelines, budgets, and resource allocation for projects.
  • Assigning tasks to team members and coordinating their efforts.
  • Monitoring project progress and ensuring adherence to timelines and budgets.
  • Mentoring and supporting team members to enhance performance and productivity.
